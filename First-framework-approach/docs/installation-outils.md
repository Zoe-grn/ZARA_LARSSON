# 🛠️ Installation des outils

## Extensions VS Code à installer

### 1. Live Share (pour la collaboration)

- Rechercher "Live Share" dans les extensions VS Code
- Permet de travailler à plusieurs en temps réel
- **Essentiel** pour ce projet en équipe

### 2. Figma (pour accéder à la maquette directement dans VS Code)

- Rechercher "Figma" dans les extensions VS Code
- Installer l'extension officielle **"Figma"** par Figma
- Permet de visualiser et inspecter la maquette Figma directement dans votre éditeur

### 3. Live Server (optionnel, pour prévisualiser le projet)

- Rechercher "Live Server" dans les extensions VS Code
- Permet de voir votre site en temps réel dans le navigateur
- Utile pour tester vos modifications instantanément

## Vérification de l'installation

Une fois les extensions installées, vous devriez voir leurs icônes dans la barre latérale gauche de VS Code.
