# 🧩 Composants HTML

## Concept

Ce dossier contient des **composants HTML réutilisables**, exactement comme les composants dans Figma !

## 💡 Parallèle avec Figma

Dans **Figma**, vous créez un composant "Button" que vous pouvez :

- Réutiliser dans plusieurs frames
- Dupliquer avec du contenu différent
- Modifier une fois pour mettre à jour toutes les instances

Ici, c'est **pareil** :

- Chaque fichier `.html` est un composant réutilisable
- Vous copiez-collez le contenu dans vos pages
- Vous adaptez le contenu selon vos besoins

## 📋 Composants disponibles

- **`header.html`** - En-tête du site (navigation, logo)
- **`footer.html`** - Pied de page
- **`hero.html`** - Section d'accroche principale
- **`card.html`** - Carte réutilisable (pour listes, grilles)
- **`button.html`** - Boutons (primaire, secondaire, ghost)

## 🚀 Comment utiliser

1. **Ouvrir** le fichier du composant (ex: `header.html`)
2. **Copier** tout le contenu HTML (sauf les commentaires d'explication)
3. **Coller** dans votre page (`index.html`, `pages/...`)
4. **Adapter** le contenu (textes, images, liens) selon votre maquette Figma
